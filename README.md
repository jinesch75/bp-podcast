# Biergerpakt Podcast

A small website for the Biergerpakt Podcast: listen to each episode, follow the
script as it highlights along with the audio, and take a 5-question quiz to earn
a personal certificate.

## Episodes
1. MyGuichet.lu – Your Online Door to Luxembourg's Administration
2. Your Health, Online – The DSP and How the CNS Pays You Back
3. Eltereforum – Strong Parents, Strong Children
4. Volunteering in Luxembourg – Give the Time You Want

## How it works
- Open `index.html` in a browser. It's a static site — no server needed.
- The script highlighting is driven by timestamps in `episodes_data.js`,
  generated by forced-aligning each MP3 against its script.
- Each quiz requires all five answers correct; wrong answers reappear at the end
  until answered correctly, then a printable certificate is shown.

## Files
- `index.html` – page structure and styling
- `app.js` – application logic (player sync, quiz, certificate)
- `episodes_data.js` – episode metadata, script segments with timestamps, quiz questions
- `podcast_*.mp3` – the audio for each episode
- `podcast_script_*.md` – the source scripts
- `gov-light.svg`, `favicon.svg` – branding assets
